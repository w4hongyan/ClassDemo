# 小学信息课互动演示游戏

> 专为小学信息课程设计的HTML5互动学习演示游戏集合

## 📁 项目结构

```
ClassDemo/
├── 五年级/
│   ├── 二分法猜数.html          # 二分法算法演示与猜数游戏
│   └── 凯撒密码.html            # 密码学基础与凯撒密码演示
├── 四年级/
│   ├── 校验码.html              # 数据校验原理与编码演示
│   └── 电报.html                # 莫尔斯电码与电报原理
└── README.md                     # 项目说明文档（本文件）
```

## 🎯 项目简介

本项目包含四个精心设计的信息技术演示游戏，旨在通过互动体验帮助小学生理解信息技术中的核心概念。每个游戏都结合了理论学习和实践操作，让抽象的概念变得直观易懂。

## 📚 详细文件说明

### 🔍 五年级

#### 1. 二分法猜数.html
- **主要功能**：二分法算法演示与互动猜数游戏
- **教学目标**：
  - 理解二分查找算法的基本原理
  - 体验算法效率的概念
  - 培养逻辑思维能力
- **特色**：
  - 可视化算法执行过程
  - 实时显示搜索范围
  - 随机生成目标数字
  - 动态统计比较次数
- **适用年级**：五年级

#### 2. 凯撒密码.html  
- **主要功能**：凯撒密码加解密演示与互动游戏
- **教学目标**：
  - 了解密码学基础概念
  - 理解加密和解密过程
  - 体验古典密码的原理
- **特色**：
  - 支持自定义偏移量
  - 实时加解密演示
  - 包含小游戏测试
  - 完整的字母表对照
- **适用年级**：五年级

### 🔧 四年级

#### 3. 校验码.html
- **主要功能**：数据校验原理演示与编码解码游戏
- **教学目标**：
  - 理解数据校验的重要性
  - 掌握简单的校验码计算
  - 体验错误检测机制
- **特色**：
  - 可视化校验过程
  - 错误数据检测演示
  - 交互式编码解码练习
  - 即时反馈系统
- **适用年级**：四年级

#### 4. 电报.html
- **主要功能**：电报与莫尔斯电码学习演示
- **教学目标**：
  - 了解电报通信的历史
  - 掌握莫尔斯电码基础
  - 体验早期通信技术
- **特色**：
  - 虚拟电报机界面
  - 真实音频效果
  - SOS求救信号演示
  - 完整的电码对照表
  - 互动测试游戏
- **适用年级**：四年级

## 🚀 使用指南

### 快速开始

1. **直接运行**：双击任意HTML文件即可在浏览器中打开
2. **本地服务器**：推荐使用HTTP服务器运行以获得最佳体验
   ```bash
   # Python方式
   python -m http.server 8080
   
   # 访问地址
   http://localhost:8080
   ```

### 浏览器兼容性

- ✅ Chrome 60+
- ✅ Firefox 60+ 
- ✅ Safari 12+
- ✅ Edge 80+
- ⚠️ 需要启用JavaScript

### 系统要求

- **操作系统**：Windows 7+, macOS 10.12+, Linux
- **网络**：可选（仅需加载字体资源）
- **硬件**：无特殊要求，音频功能需要扬声器/耳机

## 👨‍🏫 教学建议

### 课堂使用建议

1. **课前准备**
   - 提前测试所有功能
   - 准备备用教学方案
   - 检查音频设备正常

2. **授课流程**
   - 理论讲解（10-15分钟）
   - 演示操作（10分钟）
   - 学生实践（15-20分钟）
   - 总结讨论（5-10分钟）

3. **互动指导**
   - 鼓励学生自主探索
   - 适时提供提示和引导
   - 组织小组合作学习

### 年级适配建议

- **校验码.html**：强调实际应用场景，概念讲解宜简不宜深

#### 五年级重点  
- **二分法猜数.html**：重点算法思想，图形化展示过程
- **凯撒密码.html**：扩展古代密码知识，激发兴趣

### 拓展活动建议

1. **编程启蒙**：引导学生思考游戏背后的逻辑
2. **历史探索**：结合电报、密码学历史故事
3. **实践创新**：鼓励学生设计简单的编码方案
4. **团队协作**：组织编码解码竞赛活动

## ⚠️ 注意事项

### 技术注意事项

1. **音频权限**：首次访问时浏览器可能请求音频权限，请选择"允许"
2. **屏幕适配**：建议在1024x768及以上分辨率下使用
3. **网络要求**：首次加载可能需要网络连接获取字体资源

### 教学注意事项

1. **学生差异**：根据学生基础调整讲解深度
2. **时间控制**：每节课建议控制在45分钟内
3. **设备限制**：如设备不足可采用分组轮换方式

### 故障排除

| 问题 | 解决方法 |
|------|----------|
| 音频无声 | 检查浏览器音频权限设置 |
| 页面显示异常 | 尝试刷新页面或更换浏览器 |
| 游戏无法开始 | 确认JavaScript已启用 |
| 字体显示异常 | 检查网络连接或刷新页面 |
